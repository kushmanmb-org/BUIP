# Privacy Policy

## Overview

This Privacy Policy describes how the Bitcoin Unlimited Improvement Proposal (BUIP) repository handles information and protects the privacy of contributors and users.

## Information Collection

### Public Information

- All contributions to this repository (commits, issues, pull requests, comments) are public and permanently recorded in the Git history.
- Author information (name, email) included in Git commits is public.
- GitHub usernames and profile information are publicly visible.

### What We Don't Collect

- We do not collect personal information beyond what is publicly available on GitHub.
- We do not use cookies or tracking technologies.
- We do not sell or share personal information with third parties.

## Data Usage

Information in this repository is used solely for:

- Maintaining the BUIP documentation archive
- Facilitating collaboration among Bitcoin Unlimited contributors
- Version control and historical record-keeping
- Public review and voting on proposals

## Your Rights

### Access and Control

- You control your GitHub profile information through your GitHub account settings.
- Git commit history is immutable by design for integrity purposes.
- You can request removal of sensitive information accidentally committed (e.g., API keys, passwords).

### Removing Contributions

- Once merged, contributions become part of the permanent historical record.
- You can request your GitHub account be disassociated from contributions, but the content may remain for historical accuracy.

## Third-Party Services

This repository uses GitHub, which has its own privacy policy:
- [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

## Data Security

- All data is hosted on GitHub's secure infrastructure.
- Access to repository management is limited to authorized maintainers.
- We follow security best practices for repository management.
- All communications are encrypted via HTTPS.

## CI/CD and Automated Workflows

- GitHub Actions workflows may process repository contents.
- Workflows run on self-hosted runners with restricted access.
- Workflow logs are stored according to GitHub's retention policies.
- No personal data is collected beyond what's in the repository.

## Contact

For privacy concerns or to request removal of sensitive information:

1. Open an issue in this repository with the label "privacy"
2. Contact the repository maintainers directly
3. Submit a pull request to remove the sensitive content

## Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be committed to this repository with appropriate version history.

## Compliance

This project complies with:

- General Data Protection Regulation (GDPR) principles
- GitHub's Terms of Service and Privacy Policy
- Open source collaboration best practices

## Data Retention

- Repository data is retained indefinitely for historical purposes.
- Git history is permanent by design.
- Issue and PR discussions are retained as long as the repository exists.
- Workflow logs follow GitHub's standard retention policies (typically 90 days).

## Transparency

This is an open-source project, and all activities are public and transparent. We believe in:

- Open collaboration
- Public decision-making
- Transparent governance
- Community-driven development

## Children's Privacy

This repository is not intended for use by children under 13. We do not knowingly collect information from children under 13. GitHub's Terms of Service require users to be at least 13 years old.

## International Data Transfers

Data in this repository is stored on GitHub's servers, which may be located in various countries. By contributing to this repository, you consent to international data transfer as governed by GitHub's privacy policy.
